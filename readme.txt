-- Eksplodert datamaskin --
###########################

Programmet (java) og skriptene brukes på "eksplodert datamaskin" i utstillingen Abels skissebok på Vitenfabrikken. Maskinen kjører et oppsett med Ubuntu og Fluxbox.

## Nyttige kommandoer når man er innlogget på maskinen:
x11vnc

## Oppdateringer i skriptene
Oppdateringer kan gjøres på hvilken som helst maskin. Endringer legges inn på Github og de hentes så ned på selve maskinen via SSH.

## Oppdatere skriptene på selve maskinen:

- Åpne SSH-forbindelse til maskinen
(Du skal nå stå i home-mappa til vitenfabrikken, /home/vitenfabrikken/)
- Kjør git pull origin master
- Alt er nå hentet fra Github i siste versjon
